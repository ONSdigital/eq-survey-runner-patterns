---
title: The eQ Grid
---
A flexible grid based on __12__ columns, each column has a percentage calculated width and the margins (gutters) are fixed for each breakpoint.

The baseline grid is based on multiples of 8px.
### 600 - 1000+ width
Gutters: 24px
### 301 - 600 width
Gutters: 16px
### <300 width
GUtters: 16px
